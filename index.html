Automation / Clips / Arrangement

Automation operators

Operators target module parameters over clip-local beats: step, linear ramps, and LFO curves. A clip can carry MIDI notes plus any number of automation lanes.

Clip session view

Clip slots launch/stop in channels quantized to the global clock, Ableton-style. Clips contain a length in bars, MIDI/control events, velocity and parameter automation.

Arrangement timeline

Clips can also be placed at absolute global beats on a timeline with play, pause, skip and loop regions in the core model.
